## 🚀 How to Deploy on GitHub Pages (Free Hosting)

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) → **Sign in** (or create a free account)
2. Click **"New repository"** (the green button)
3. Name it: `sivakumar-portfolio` (or any name you like)
4. Set it to **Public**
5. Click **"Create repository"**

### Step 2 — Upload the File
1. In your new repo, click **"Add file" → "Upload files"**
2. Drag and drop **`index.html`** from this folder
3. Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. In your repo, go to **Settings** → **Pages** (left sidebar)
2. Under **"Source"**, select: **`main`** branch → **`/ (root)`** folder
3. Click **"Save"**
4. Wait ~60 seconds, then your site will be live at:
   `https://YOUR-USERNAME.github.io/sivakumar-portfolio/`
